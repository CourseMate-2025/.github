## LLM 기반 학업 지원 서비스 - 코스메이트( Course-Mate )


🙋‍♀️ About Course Mate
Course-Mate(코스메이트)는 가천대학교 컴퓨터공학과(2019~2025학년도 입학생)를 대상으로 하는 AI 기반 학업 지원 서비스이다.

대학생 개개인의 이수 내역과 목표 직무를 기반으로 학기별 과목 추천 로드맵을 제공하고, RAG 기반 생성형 AI를 통해 과목 관련 질의응답(Q&A) 서비스를 제공한다.

이를 통해 단순한 강의계획서를 넘어, 학생 수준에 맞춘 실질적이고 이해 가능한 수업 정보를 전달한다.
